Client Logos

📁 Logo Upload Instructions

  • Upload client logos to: /images/clients/
  • Recommended format: SVG or PNG with transparency
  • Recommended size: 200px width
  • 4 slots with rotating logos - add multiple per slot

Custom Fonts

📁 Font Upload Instructions

  • Create a /fonts/ folder in your root directory
  • Upload your font files:
    • PPNeueMontreal-Book.woff
    • PPNeueMontreal-Book.woff2 (optional, for better compression)
    • PPNeueMontreal-Book.ttf (fallback)
  • The site will automatically use PP Neue Montreal for all text except monospace headers
  • Monospace text (headers/labels) uses Geist Mono

✅ Current Font Setup

FTP Setup Instructions

📁 Folder Structure

Create the following folder structure on your FTP:

  • / (root)
    • index.html - Main portfolio file
    • admin.html - This admin panel
    • /data/
      • site-content.json - Generated content file
    • /fonts/
      • PPNeueMontreal-Book.woff
      • PPNeueMontreal-Book.woff2
      • PPNeueMontreal-Book.ttf
    • /images/
      • /projects/ - Project images
      • /clients/ - Client logos

🔧 How to Use

  1. Upload all files to your FTP server
  2. Upload images to respective folders
  3. Access yourdomain.com/admin.html
  4. Fill in all content fields
  5. Click "Generate JSON" to create the content file
  6. Save the JSON as site-content.json in the /data/ folder
  7. Your portfolio at yourdomain.com will auto-update

💡 Tips

  • Always backup your site-content.json before making changes
  • Test locally first by opening files in your browser
  • Use relative paths for images: images/projects/project1.jpg
  • SVG logos work best for crisp display at any size
  • Client logos rotate automatically every 3-5 seconds
